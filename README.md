# DAX Depo – Advanced Calculations Using DAX in Power BI

## 📌 Project Overview
This project demonstrates advanced analytical calculations using **DAX (Data Analysis Expressions)** in **Power BI**.  
The report is built using a Sales and Returns data model and focuses on implementing calculated columns, measures, filter context functions, and time intelligence analysis using Matrix visuals.

---

# 📂 Dataset Tables Used

- `Sales_Fact`
- `Returns_Fact`
- `Customer_Dim`
- `Product_Dim`
- `Date_Dim`
- `Region_Dim`

---

# 🎯 Objectives

- Build a proper data model using relationships
- Create calculated columns and measures using DAX
- Perform sales and return analysis
- Implement time intelligence calculations
- Use matrix visuals for analytical reporting
- Understand filter context behavior in Power BI

---

# 🛠 Features Implemented

## ✅ Calculated Columns
